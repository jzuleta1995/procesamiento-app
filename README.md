# Procesamiento-app
Aplicación que consta de una api desarrollada en spring boot encargada con una arquitectura de 3 capas - Dao - Service - Controller con un patrón de diseño Facade.
El backend cuenta con 3 endpoints:
1 - Listar clientes
2 - Crear clientes
3 - Actualizar el estado de cliente

El front end se ecuentra desarrollado en html, js y css con librerías tales como jquery y bootstrap. La arquitetura es basada en mvc.
